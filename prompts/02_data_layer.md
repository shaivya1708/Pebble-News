# Step 2: Data Layer

## Prompt

```
Set up news data handling:

1. Connect to NewsAPI.org to fetch news articles
2. Store user data locally (login info, favorites, preferences)
3. Each user should have their own separate favorites and settings
4. News should be from the last 7 days
5. Support 4 news categories: Technology, Business, World, Economy
```

## What You Get

- ✅ App can fetch news from internet
- ✅ User data saved on device
- ✅ Each user has separate saved articles
