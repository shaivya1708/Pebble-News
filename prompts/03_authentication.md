# Step 3: Authentication

## Prompt

```
Create login and registration screens:

1. Login Screen:
   - Email and password fields
   - Login button
   - Link to register page
   - Show error if wrong credentials

2. Register Screen:
   - Email and password fields
   - Password must be at least 6 characters
   - Show error if email already exists

3. Auto-login:
   - If user already logged in, skip to home screen
   - Hide toolbar on login/register pages
```

## What You Get

- ✅ Login screen
- ✅ Register screen  
- ✅ Returning users go straight to home

## User Flow

```
App Opens → Already logged in? → Yes → Home Screen
                               → No  → Login Screen
```
